# プライバシーポリシー / Privacy Policy

## アプリケーション名 / Application Name
MultiNode Sync Builder (SynArrowz)

## 個人情報の取り扱いについて / Handling of Personal Information
本アプリケーションは、ユーザーの個人情報を収集、保存、または第三者へ送信することは一切ありません。
This application does not collect, save, or transmit any user's personal information to third parties.

## データの利用 / Data Usage
本アプリケーションで使用されるノードやフィールド、テンプレートなどのデータは、すべてお客様のデバイス内にのみ保存され、外部サーバーへ送信されることはありません。
All data used in this application, including nodes, fields, and templates, is stored exclusively on your device and is never transmitted to external servers.

If you have any questions regarding this application, please open an issue on our [GitHub Issues page](https://github.com/SynArrowz/AppSupport/issues).
